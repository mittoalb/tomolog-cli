===========
tomolog-cli
===========

Documentation: at `tomolog cli <https://tomologcli.readthedocs.io/en/latest/>`_

Every run appends an entry to ``~/.tomolog`` (YAML) recording the date, GUP/proposal number, username, badge ID, beamline, HDF5 file path, and Google Slides presentation URL. This history file can be queried by other tools to retrieve the presentation URL for a given proposal number.
